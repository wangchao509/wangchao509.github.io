# 磁盘

问题现象：
当磁盘容量不足的时候，应用时常会抛出如下的异常信息：

java.io.IOException: 磁盘空间不足

排查思路
利用df命令查看磁盘状态

df -h
h表示以human理解的方式展示，会除以1024kb，换算成M

利用du命令查看文件夹大小

du -hs *
h表示以human理解的方式展示，会除以1024kb，s表示sumary，就是统计文件夹大小的意思

利用 ls 查看文件大小

ls -lh h同上，l表示以长列格式展示
